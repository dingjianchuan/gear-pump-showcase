# Gear Pump Showcase

Static GitHub Pages site for a micro gear pump SolidWorks model showcase.

Open the published site with GitHub Pages. The page loads the exported SolidWorks STL assembly from `assets/` and renders it with WebGL.

Required files:

- `index.html`
- `styles.css`
- `app.js`
- `model-data.js`
- `assets/`
